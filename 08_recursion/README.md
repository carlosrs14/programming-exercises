# Recursion

Exercises solved recursively in C.

## Exercises

- **1. Recursive Factorial:** Calculates the factorial of a non-negative integer N recursively.
- **2. Recursive Fibonacci:** Finds the N-th term of the Fibonacci sequence using recursion.
- **3. Towers of Hanoi:** Solves the Towers of Hanoi puzzle for N disks.
- **4. Recursive Binary Search:** Searches for an element in a sorted array by recursively halving the search space.
- **5. Recursive Digit Sum:** Calculates the sum of digits of an integer recursively.
