# Structures (Structs)

Exercises to group variables of different types using structs.

## Exercises

- **1. Student Average:** Stores student details and calculates the average grade using a Student struct.
- **2. Distance Between Points:** Calculates the Euclidean distance between two 2D points stored as structs.
- **3. Complex Numbers Addition:** Adds two complex numbers represented as structs.
- **4. Product Inventory:** Calculates the total inventory value for N products stored as Product structs.
- **5. Date Validation:** Validates a calendar date using a Date struct.
- **6. Time Difference:** Calculates the elapsed time in HH:MM:SS between two times using Time structs.
- **7. Highest Salary:** Finds the employee with the highest salary from a list using an Employee struct.
- **8. Fraction Multiplication:** Multiplies two fractions represented as structs and simplifies the result.
- **9. Contact Search:** Searches for a contact by name in a list of Contact structs.
- **10. Point in Rectangle:** Checks if a point is inside a rectangle using Point and Rectangle structs.
