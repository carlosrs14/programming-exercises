# Functions

Exercises to practice modularizing code by writing and utilizing functions.

## Exercises

- **1. Happy new year:** Count 10 to 0 and print happy new year.
- **2. Areas of Shapes:** Calculates the area of a square, rectangle, or triangle using separate helper functions.
- **3. Next Prime:** Finds the smallest prime number strictly greater than N using an is_prime helper function.
- **4. Decimal to Binary:** Converts a decimal number to its binary representation.
- **5. Power Function:** Calculates the power of a base float to an integer exponent without using math.h.
- **6. Least Common Multiple (LCM):** Calculates the LCM of two integers using helper functions for GCD and LCM.
- **7. Digit at Position K:** Returns the digit at the K-th position from right to left of an integer N.
- **8. Range Sum:** Returns the sum of all integers in the closed range [a, b].
- **9. Count Leap Years in Range:** Counts how many leap years exist in an inclusive range of years.
- **10. Combinations C(n, k):** Calculates combinations C(n, k) = n! / (k! * (n-k)!) using a helper factorial function.
