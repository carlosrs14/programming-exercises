# Sorting

Classical algorithms to sort numeric arrays in C.

## Exercises

- **1. Bubble Sort:** Implements the classical bubble sort algorithm iteratively.
- **2. Selection Sort:** Repeatedly finds the minimum element and places it at the correct index.
- **3. Insertion Sort:** Inserts each element into its sorted position relative to preceding elements.
- **4. Merge Sort:** A recursive divide-and-conquer algorithm that halves the array, sorts, and merges.
- **5. Quick Sort:** Partitions the array based on a pivot and recursively sorts the sub-arrays.
