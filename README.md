# C
learn C by training

## Topics

1. **[01_data_types](./01_data_types)**: Basic variable declarations, `scanf` input, and `printf` output formatting.
2. **[02_conditionals](./02_conditionals)**: Decision-making with `if`, `else if`, `else`, and `switch` statements.
3. **[03_loops](./03_loops)**: Repeated executions using `for`, `while`, and `do-while` constructs.
4. **[04_functions](./04_functions)**: Modularizing code, parameter passing, and functional units.
5. **[05_pointers](./05_pointers)**: Pointer arithmetic, pass-by-reference, and dynamic memory allocation.
6. **[06_structs](./06_structs)**: User-defined composite data structures.
7. **[07_sorting](./07_sorting)**: Classic sorting algorithms.
8. **[08_recursion](./08_recursion)**: Exercises solved using recursive functions.
9. **[09_gui](./09_gui)**: Graphical user interface (GUI) learning exercises.
10. **[10_projects](./10_projects)**: Complete console-based games and tools.

## Build and Run

Use `gcc`:

```bash
gcc file_name.c
./a.out
```
