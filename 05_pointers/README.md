# Pointers

Exercises to understand pointer arithmetic, pass by reference, and dynamic memory management.

## Exercises

- **1. Swap by Reference:** Swaps two integers using pointers.
- **2. Sum Array with Pointers:** Sums the elements of an array by traversing it using pointer arithmetic.
- **3. Min and Max of Array:** Finds the minimum and maximum elements in an array, returning them via pointers.
- **4. Reverse Array In-Place:** Reverses an array in-place using two pointers advancing from opposite ends.
- **5. String Length:** Calculates the length of a string by iterating with a pointer until the null terminator is reached.
- **6. Copy String:** Copies a source string to a destination buffer using pointers.
- **7. Count Vowels and Consonants:** Counts the vowels and consonants in a string using pointers.
- **8. Search Element with Pointers:** Searches for an integer in an array and returns a pointer to it.
- **9. Concatenate Strings:** Concatenates two strings using pointers.
- **10. Dynamic Array of Multiples:** Dynamically allocates an array, fills it with multiples of a number, and frees the memory.
