# Loops

Exercises to practice loops and repetition statements (for, while, do-while).

## Exercises

- **1. Multiplication Table:** Prints the multiplication table of a given number from 1 to 10.
- **2. Factorial of a Number:** Calculates the factorial N! of a non-negative integer N iteratively.
- **3. Fibonacci Sequence:** Prints the first N terms of the Fibonacci sequence.
- **4. Primes in Range:** Prints all prime numbers less than or equal to an integer N.
- **5. Sum of Digits:** Sums the individual digits of an integer iteratively.
- **6. Reverse a Number:** Reverses the digits of an integer, maintaining the sign.
- **7. Count Digits:** Counts how many digits make up an integer.
- **8. Pyramid of Asterisks:** Draws a symmetric pyramid of asterisks of height N in the console.
- **9. Greatest Common Divisor (GCD):** Calculates the GCD of two integers using the Euclidean algorithm.
- **10. Perfect Number:** Determines if an integer is equal to the sum of its proper positive divisors.
