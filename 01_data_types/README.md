# Data Types

Exercises to practice variable declaration, basic standard input with scanf, and output with printf for primitive types (int, float, char, etc.).

## Exercises

- **1. Sum of Integers:** Reads two integers from standard input and displays their sum.
- **2. Area of a Circle:** Reads the radius of a circle and calculates its area using 3.14159 as the value of PI.
- **3. Celsius to Fahrenheit:** Reads a temperature in Celsius and converts it to Fahrenheit (F = C * 1.8 + 32).
- **4. Arithmetic Operators:** Reads two integers and shows their sum, difference, product, quotient, and remainder.
- **5. Average of Three Grades:** Reads three grades as floats and calculates their simple average.
- **6. Time Conversion:** Converts a given amount of seconds to hours, minutes, and seconds in HH:MM:SS format.
- **7. Swap Without Helper Variable:** Swaps the value of two integer variables using only arithmetic operators.
- **8. Distance Between Two Points:** Calculates the Euclidean distance between two points in a 2D plane (x1, y1) and (x2, y2).
- **9. Currency Converter:** Converts US Dollars (USD) to Euros (EUR) and Mexican Pesos (MXN).
- **10. Rectangle Area and Perimeter:** Reads the width and height of a rectangle and calculates its area and perimeter.
